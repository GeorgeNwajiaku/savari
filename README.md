# Savari — African Tailoring, Delivered Worldwide

Premium marketplace connecting skilled Nigerian & African tailors with customers worldwide.

## Features
- Browse catalog by outfit category
- View & select tailors by rating and specialty
- Enter and save body measurements (multiple profiles)
- Place custom orders with fabric/colour/add-on options
- Upload inspiration photos
- Track orders from production to delivery
- Admin dashboard
